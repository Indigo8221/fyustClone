# 福耀科技大学校园网站

这是一个基于 Vue 3 + TypeScript + Vite 构建的现代化校园网站前端项目。

## 技术栈

- Vue 3
- TypeScript
- Vite
- Vue Router
- CSS3 动画


## 部署

项目已配置 GitHub Actions 自动部署到 GitHub Pages。

访问地址：https://indigo8221.github.io/fyustClone/

## 项目结构

```
src/
├── components/     # 组件
├── views/         # 页面
├── router/        # 路由配置
├── assets/        # 静态资源
└── main.ts        # 入口文件
```
